# Virtual Piano using Python and OpenCV

This project is a Virtual Piano built using Python and OpenCV.

Users can play piano keys using keyboard or mouse.

Technologies Used

Python
OpenCV
Computer Vision
Pygame

Features

Real-time piano sound
Interactive piano keys
Computer vision integration

How to Run

pip install -r requirements.txt

python main.py
